# `Animator` Class

The `Animator` class mirrors the Unity Animator component. It is accessed via the `animator` property of a `GameObject`.

## Methods

### `SetTrigger(triggerName)`
Sets an animator trigger.

### `SetFloat(name, value)`
Sets a float parameter.

### `SetInteger(name, value)`
Sets an integer parameter.

### `SetBool(name, value)`
Sets a boolean parameter.

### `Play(stateName)`
Plays the specified animator state.

### `SetLayerWeight(layerIndex, weight)`
Sets the weight of the animator layer at the specified index.
